## 2. 문제 정의

### 사용자가 겪는 문제

**성분 이해의 어려움**
- 화장품 성분명이 전문 용어로 되어 있어 일반인이 이해하기 어려움
- "나이아신아마이드", "히알루론산" 같은 성분이 무슨 역할인지, 내 피부에 맞는지 알기 힘듦
- 기존 앱/서비스들은 설명이 딱딱하거나 정보가 과다하여 오히려 혼란스러움

**화장법 적용의 어려움**
- 베이스 메이크업 시 화장이 뜨거나 몽글몽글 올라오는 문제 발생
- 유튜브 화장법을 따라해도 피부 타입이 달라 나에게 맞지 않음
- 어떤 성분의 제품을 어떻게 사용해야 하는지 모름

### 우리의 해결 방향
- 전문 용어 없이 쉬운 말로 설명
- 핵심 정보만 간결하게 전달
- "왜 좋은지", "언제 쓰면 좋은지" 실용적인 정보 제공
- **피부 타입별 맞춤 화장법 팁 제공**

| 뷰티 덕후 | 스킨케어에 관심 많은 사용자 | 상세 정보, 성분 비교 |

## 4. 핵심 기능

### 4.1 성분 검색
사용자가 원하는 방식으로 성분 정보에 접근할 수 있도록 다양한 검색 방법 제공

- **텍스트 검색**: 성분명을 직접 입력하여 검색
- **바코드 스캔**: 제품 바코드를 스캔하여 전체 성분 목록 확인
- **사진 인식**: 성분표 사진을 찍으면 자동으로 성분 인식

### 4.2 시각적 성분 설명
복잡한 정보를 한눈에 파악할 수 있도록 시각화

- **아이콘**: 성분의 주요 효능을 직관적인 아이콘으로 표시
- **등급 표시**: 안전성, 자극도 등을 등급으로 표현
- **컬러 코딩**: 좋음/주의/위험 등을 색상으로 구분

### 4.3 실용 정보 제공
단순한 설명을 넘어 실제 사용에 도움되는 정보 제공

- **왜 좋은지**: 이 성분이 피부에 어떤 이점을 주는지
- **언제 사용하면 좋은지**: 아침/저녁, 계절, 피부 상태별 추천 시기
- **함께 쓰면 좋은/피해야 할 성분**: 성분 간 궁합 정보

### 4.4 피부 타입별 화장법 팁
성분 정보와 연계하여 내 피부에 맞는 화장법 제공

**베이스 메이크업 (중점)**
- 스킨케어 순서: 피부 타입별 최적의 스킨케어 루틴
- 프라이머 사용법: 지성/건성/복합성에 따른 프라이머 선택과 도포 방법
- 파운데이션 밀착법: 화장이 뜨지 않고 오래 유지되는 방법

**색조 메이크업 추천**
- 피부톤/타입에 맞는 색조 제품 추천
- 계절별, 상황별 메이크업 팁

**콘텐츠 형태**
- 텍스트 팁: 핵심만 담은 간결한 설명
- 단계별 이미지: 시각적으로 따라하기 쉬운 가이드
- 짧은 영상 클립: 실제 적용 방법을 보여주는 숏폼 영상

**성분 연계 팁**
- "이 성분이 든 제품은 이렇게 사용하세요"
- 성분 궁합에 따른 레이어링 순서 안내

## 5. 부가 기능

### 5.1 내 피부 프로필
- 피부 타입 설정 (건성/지성/복합성/민감성)
- 피부 고민 선택 (모공, 주름, 색소침착, 트러블 등)
- 설정에 따른 맞춤 정보 및 추천 제공

### 5.2 피해야 할 성분 알림
- 개인별 피해야 할 성분 목록 저장
- 제품 검색 시 해당 성분 포함 여부 자동 알림
- 알레르기/자극 이력 기록

### 5.3 즐겨찾기
- 자주 찾는 성분 저장
- 관심 있는 제품 저장
- 나만의 성분 노트 작성

### 5.4 커뮤니티
- 사용자 리뷰 및 경험 공유
- 피부 타입별 추천 토론
- 성분 관련 Q&A

## 6. 차별화 포인트

### 경쟁 서비스 비교

| 서비스 | 특징 | 한계 |
|--------|------|------|
| 화해 | 성분 분석 + 리뷰 | 성분 정보만, 화장법 없음 |
| 유튜브 | 다양한 화장법 영상 | 개인화 불가, 피부 타입별 맞춤 어려움 |
| EWG Skin Deep | 성분 안전성 등급 | 영어, 화장법 없음 |

### 우리 서비스의 차별점

| 기존 서비스 | 우리 서비스 |
|-------------|-------------|
| 전문 용어 위주의 딱딱한 설명 | 전문용어 없는 쉽고 친근한 설명 |
| 정보 나열식 구성 | 실용적인 사용 팁 중심 |
| 복잡하고 정보 과다한 UI | 예쁘고 직관적인 미니멀 UI |
| 출처 불분명한 데이터 | 정확하고 신뢰할 수 있는 데이터 |
| 성분 정보만 제공 | **성분 + 피부 타입별 화장법 팁** |
| 일반적인 화장법 | **내 피부에 맞는 맞춤 화장법** |

## 8. 시장 분석

### 시장 기회

**메이크업 클래스 붐**
- "메이크업 클래스 다녀왔더니 인생이 바뀌었다" 류의 인스타 릴스 급증
- 사람들이 "제대로 된 화장법"을 배우고 싶어함
- 유튜브만으로는 부족함을 느끼고, 개인 맞춤 피드백을 원함

**이 앱이 채우는 틈새**

| 메이크업 클래스 | 이 앱 |
|----------------|-------|
| 수십만원 비용 | 무료 또는 저렴한 구독료 |
| 1회성, 오프라인 | 상시 이용, 언제 어디서나 |
| 시간/장소 제약 | 필요할 때마다 바로 확인 |

> "클래스 가기엔 부담스럽고, 유튜브론 안 되는" 사람들을 위한 서비스
> 
> 클래스 다녀온 사람에겐 **복습 도구**, 못 간 사람에겐 **대안**

### 리스크 및 대응

| 리스크 | 대응 방향 |
|--------|-----------|
| 화장법 콘텐츠 제작 비용 | 초기엔 텍스트+이미지 중심, 영상은 점진적 확대 |
| 화해가 비슷한 기능 추가 가능 | 화장법 특화로 선점, 깊이 있는 콘텐츠로 차별화 |
| 초기 사용자 확보 어려움 | 커뮤니티 활성화, SNS 바이럴 전략 |

## 9. 실행 전략 (권장)

### MVP (최소 기능 제품) 우선순위

처음부터 모든 기능을 만들지 말고, 단계적으로 확장

| 단계 | 기능 | 목표 |
|------|------|------|
| 1단계 | 성분 사전 + 피부 타입별 화장법 팁 (텍스트) | 핵심 가치 검증 |
| 2단계 | 이미지 가이드 + 피부 프로필 | 사용자 경험 강화 |
| 3단계 | 바코드/사진 스캔 + 영상 콘텐츠 | 편의성 확대 |
| 4단계 | 커뮤니티 + 알림 기능 | 리텐션 강화 |

### 콘텐츠 확보 전략

**초기 콘텐츠**
- 피부과 전문의, 메이크업 아티스트 자문
- 검증된 출처의 성분 정보 활용 (EWG, 대한화장품협회 등)
- 피부 타입별 화장법은 전문가 감수 후 제작

**확장 콘텐츠**
- 뷰티 크리에이터/인플루언서 협업
- 사용자 제작 콘텐츠 (UGC) 활용
- 브랜드 협찬 콘텐츠 (수익화 연계)

### 사용자 확보 전략

| 채널 | 방법 |
|------|------|
| 인스타그램 | "화장이 뜨는 이유" 등 공감 콘텐츠로 유입 |
| 틱톡/릴스 | 피부 타입별 before-after 숏폼 |
| 네이버 카페 | 뷰티 커뮤니티 타겟 마케팅 |
| 입소문 | 친구 초대 시 프리미엄 기능 제공 |

### 수익 모델 (참고)

| 모델 | 설명 |
|------|------|
| 프리미엄 구독 | 상세 화장법 영상, 1:1 맞춤 추천 |
| 광고 | 화장품 브랜드 네이티브 광고 |
| 제휴 커머스 | 추천 제품 구매 링크 수수료 |
| 클래스 연결 | 오프라인 메이크업 클래스 예약 중개 |

# 정보 구조 (Information Architecture)

---

## 1. 앱 구조 개요

```
┌─────────────────────────────────────────────────────────┐
│                      스플래시                            │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                 온보딩 (최초 1회)                        │
│  앱 소개 → 피부 타입 설정 → 회원가입/로그인              │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                      메인 화면                           │
│  ┌─────┬─────┬─────┬─────┬─────┐                       │
│  │ 홈  │검색 │스캔 │화장법│마이 │  ← 하단 탭 (5개)      │
│  └─────┴─────┴─────┴─────┴─────┘                       │
└─────────────────────────────────────────────────────────┘
```

## 2. 화면 목록

### 2.1 온보딩 플로우

| 화면 | 설명 | 비고 |
|------|------|------|
| 스플래시 | 앱 로고, 로딩 | 1~2초 |
| 온보딩 슬라이드 | 앱 소개 (3~4장) | 스킵 가능 |
| 피부 타입 설정 | 피부 타입 + 고민 선택 | 필수 |
| 회원가입 | 이메일/소셜 로그인 | 필수 |
| 로그인 | 기존 회원 로그인 | - |

### 2.2 메인 탭 화면

| 탭 | 화면명 | 설명 |
|----|--------|------|
| 홈 | 홈 피드 | 맞춤 추천 성분, 화장법 팁, 인기 콘텐츠 |
| 검색 | 성분 검색 | 텍스트로 성분/제품 검색 |
| 스캔 | 스캔 화면 | 바코드/성분표 사진 촬영 |
| 화장법 | 화장법 목록 | 피부 타입별 화장법 팁 모음 |
| 마이 | 마이페이지 | 프로필, 저장 목록, 설정 |

### 2.3 상세 화면

| 화면 | 진입 경로 | 설명 |
|------|-----------|------|
| 성분 상세 | 검색/스캔/홈 | 성분 정보 + 실용 팁 + 관련 화장법 |
| 제품 성분 목록 | 스캔 결과 | 제품 내 전체 성분 리스트 |
| 화장법 상세 | 화장법 탭/홈 | 단계별 가이드 (텍스트/이미지/영상) |

### 2.4 마이페이지 하위 화면

| 화면 | 설명 |
|------|------|
| 내 피부 프로필 | 피부 타입/고민 수정 |
| 피해야 할 성분 | 개인 블랙리스트 관리 |
| 즐겨찾기 | 저장한 성분/제품/화장법 |
| 알림 설정 | 푸시 알림 on/off |
| 계정 설정 | 로그아웃, 탈퇴 등 |

### 2.5 커뮤니티 (추후 개발)

| 화면 | 설명 |
|------|------|
| 커뮤니티 피드 | 사용자 글 목록 |
| 글 상세 | 댓글, 좋아요 |
| 글 작성 | 경험/리뷰 공유 |

## 3. 사용자 플로우

### 3.1 핵심 플로우: 성분 검색

```
홈/검색 탭
    ↓
검색어 입력 (예: "나이아신아마이드")
    ↓
검색 결과 목록
    ↓
성분 상세 페이지
  ├── 성분 설명 (시각적)
  ├── 왜 좋은지 / 언제 쓰면 좋은지
  ├── 함께 쓰면 좋은/피해야 할 성분
  └── 관련 화장법 팁 → 화장법 상세로 이동
```

### 3.2 핵심 플로우: 바코드/사진 스캔

```
스캔 탭
    ↓
카메라 실행
  ├── 바코드 스캔
  └── 성분표 사진 촬영
    ↓
제품 성분 목록
  ├── 전체 성분 리스트
  ├── 주의 성분 표시 (빨간색)
  └── 피해야 할 성분 알림 (내 블랙리스트 기준)
    ↓
개별 성분 탭 → 성분 상세 페이지
```

### 3.3 핵심 플로우: 화장법 보기

```
화장법 탭
    ↓
카테고리 선택
  ├── 베이스 메이크업
  ├── 색조 메이크업
  └── 피부 타입별
    ↓
화장법 목록
    ↓
화장법 상세
  ├── 단계별 설명
  ├── 이미지 가이드
  ├── 영상 클립 (있는 경우)
  └── 관련 성분 정보 → 성분 상세로 이동
```

## 4. 화면별 주요 요소

### 4.1 홈 피드

| 영역 | 내용 |
|------|------|
| 상단 | 인사말 + 오늘의 팁 |
| 섹션 1 | 내 피부에 맞는 추천 성분 |
| 섹션 2 | 오늘의 화장법 팁 |
| 섹션 3 | 인기 성분 / 트렌드 |

### 4.2 성분 상세 페이지

| 영역 | 내용 |
|------|------|
| 헤더 | 성분명 (한글/영문) + 아이콘 |
| 요약 | 한 줄 설명 + 등급 표시 |
| 상세 | 왜 좋은지 / 언제 쓰면 좋은지 |
| 궁합 | 함께 쓰면 좋은/피해야 할 성분 |
| 연결 | 관련 화장법 팁 카드 |
| 액션 | 즐겨찾기 / 블랙리스트 추가 |

### 4.3 화장법 상세 페이지

| 영역 | 내용 |
|------|------|
| 헤더 | 제목 + 피부 타입 태그 |
| 콘텐츠 | 단계별 설명 (텍스트 + 이미지) |
| 영상 | 숏폼 영상 (있는 경우) |
| 연결 | 관련 성분 정보 카드 |
| 액션 | 즐겨찾기 / 공유 |
